STORED FUNCTION

Definition: A stored function in PL/SQL is a named function that is stored in the database and can be reused multiple times. It performs a specific task and returns a value.

Stored functions are created once and remain in the database schema. They can be called by other PL/SQL blocks, procedures, or SQL statements (if allowed).

Syntax:

CREATE OR REPLACE FUNCTION function_name (
  param1 [mode] datatype,
  param2 [mode] datatype
) RETURN return_datatype
IS
  -- Variable declarations
BEGIN
  -- Function logic
  RETURN value;
END;

Example:

CREATE OR REPLACE FUNCTION greet_user(name IN VARCHAR2)
RETURN VARCHAR2
IS
  greeting VARCHAR2(50);
BEGIN
  greeting := 'Hello, ' || name || '!';
  RETURN greeting;
END;

Calling the Stored Function:

DECLARE
  msg VARCHAR2(50);
BEGIN
  msg := greet_user('Gagan');
  DBMS_OUTPUT.PUT_LINE(msg);
END;

Output:

Hello, Gagan!
