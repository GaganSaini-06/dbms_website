ARGUMENTS IN PROCEDURE

Definition: Arguments in a procedure are used to pass values between the calling program and the procedure. They help in making procedures dynamic and reusable.

Types of Arguments:

  • IN: Used to pass value **into** the procedure (default).
  • OUT: Used to get value **from** the procedure.
  • IN OUT: Used to pass value **in and out** of the procedure.

Syntax:

CREATE OR REPLACE PROCEDURE procedure_name(
  arg1 IN datatype,
  arg2 OUT datatype,
  arg3 IN OUT datatype)
IS
BEGIN
  -- Code using arguments
END procedure_name;

Example:

CREATE OR REPLACE PROCEDURE square_number(
  n IN NUMBER,
  square OUT NUMBER)
IS
BEGIN
  square := n * n;
END;

Calling the Procedure:

DECLARE
  x NUMBER := 5;
  y NUMBER;
BEGIN
  square_number(x, y);
  DBMS_OUTPUT.PUT_LINE('Square is: ' || y);
END;

Output:

Square is: 25
