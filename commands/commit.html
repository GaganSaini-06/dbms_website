COMMIT COMMAND

Definition: The COMMIT command is used to save all changes permanently made during the current transaction to the database.
Once committed, the changes cannot be rolled back.

Syntax:

COMMIT;

Example:

START TRANSACTION;

INSERT INTO students (Rollno, Name, Age, Class, Semester) 
VALUES (25, 'Shubman Gill', 24, 'BCA', 3);

COMMIT;

Output:

mysql> SELECT * FROM students;
+--------+-------+------+--------+
| Rollno | Name  | Age  | DeptID |
+--------+-------+------+--------+
|     18 | NONY  |   22 |      1 |
|     90 | GAGAN |   19 |      1 |
+--------+-------+------+--------+
2 rows in set (0.00 sec)

mysql> INSERT INTO students VALUES (10, 'LOVE', 21, 2);
Query OK, 1 row affected (0.07 sec)

mysql> COMMIT;
Query OK, 0 rows affected (0.00 sec)
