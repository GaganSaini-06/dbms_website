FULL OUTER JOIN

Definition: FULL OUTER JOIN returns all records when there is a match in either left or right table.
If there's no match, it returns NULL for the missing side.

Syntax:

SELECT columns FROM table1 FULL OUTER JOIN table2 ON table1.common_column = table2.common_column;
Note: MySQL doesn't directly support FULL OUTER JOIN, but we can simulate it using UNION.

Example:

SELECT s.name, m.marks FROM Students s LEFT JOIN Marks m ON s.student_id = m.student_id
UNION
SELECT s.name, m.marks FROM Students s RIGHT JOIN Marks m ON s.student_id = m.student_id;
