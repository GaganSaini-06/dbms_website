SUM() FUNCTION

Definition:The SUM() function in SQL is used to calculate the total sum of a numeric column. It adds up all the values in a column for the selected rows.

Syntax:

SELECT SUM(column_name) FROM table_name WHERE condition;

Example:

SELECT SUM(marks) FROM student_marks;
