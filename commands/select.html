SELECT COMMAND

Definition:The SELECT command in SQL is used to fetch data from a database. It retrieves one or more rows from a table.

Syntax:

SELECT column1, column2, ... FROM table_name WHERE condition;

Example:

SELECT name, age FROM students WHERE age > 18;

Output:

mysql> SELECT * FROM old_table WHERE Rollno = 18;
