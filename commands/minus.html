MINUS(DIFFERENCE)

Definition: The MINUS operator in SQL returns the rows from the first SELECT query that are not present in the second SELECT query. It removes duplicates and returns only unique rows from the first set that don't exist in the second set.

Syntax:

SELECT column1, column2 FROM table1  
MINUS  
SELECT column1, column2 FROM table2;

Example:

Query:- SELECT name FROM team1
        MINUS
        SELECT name FROM team2;

team1 = {Rohit, Virat, Gill}
team2 = {Gill, Smith, Stokes}

Result (MINUS) = {Rohit, Virat}
Shows data in team1 that is not in team2.
