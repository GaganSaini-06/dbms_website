REVOKE COMMAND

Definition: The REVOKE command is used to take back (cancel) previously granted permissions or privileges from a user in a database.

Syntax:

REVOKE privilege_name(s) ON table_name FROM 'username'@'host';

Example:

REVOKE INSERT ON students FROM 'new_user'@'localhost';

Output:

mysql>  SHOW GRANTS FOR 'new_user'@'localhost';
+--------------------------------------------------------------------------------+
| Grants for new_user@localhost                                                  |
+--------------------------------------------------------------------------------+
| GRANT USAGE ON *.* TO `new_user`@`localhost`                                   |
| GRANT SELECT, INSERT ON `bca2ndyear_4090`.`students` TO `new_user`@`localhost` |
+--------------------------------------------------------------------------------+
2 rows in set (0.00 sec)

mysql> REVOKE INSERT ON students FROM 'new_user'@'localhost';
