COMMAND

Definition:The AUTO_INCREMENT constraint is used to automatically generate a unique number when a new record is inserted into a table. It is usually applied to the primary key column.

Syntax:

CREATE TABLE table_name ( column_name INT AUTO_INCREMENT, other_column datatype, PRIMARY KEY (column_name) );

Example:

    CREATE TABLE cricket_players (
    PlayerID INT AUTO_INCREMENT,
    Name VARCHAR(50),
    Age INT,
    PRIMARY KEY (PlayerID)
);

INSERT INTO cricket_players (Name, Age) VALUES ('Virat Kohli', 35);
INSERT INTO cricket_players (Name, Age) VALUES ('Rohit Sharma', 36);
