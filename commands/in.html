COMMAND

Definition:The IN operator in SQL is used to match a column’s value against a list of multiple values. It helps you filter records where a column’s value is equal to any one of the values in the list.

Syntax:

SELECT * FROM table_name WHERE column_name IN (value1, value2, ...);

Example:

SELECT * FROM students WHERE Name IN ('Virat', 'Rohit');
