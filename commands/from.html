FROM CLAUSE

Definition:The FROM clause in SQL is used to specify the table from which the data should be retrieved. It works along with the SELECT clause and is necessary to indicate the source of the data.

Syntax:

SELECT column1, column2, ... FROM table_name;

Example:

SELECT name, age FROM students;
