CURSOR WITH FOR LOOP

Definition: A Cursor with FOR Loop simplifies cursor usage by automatically opening, fetching, and closing the cursor. No explicit `OPEN`, `FETCH`, or `CLOSE` statements are required.

Benefits: Makes code cleaner and reduces chances of cursor mismanagement.

Syntax:

FOR record_variable IN cursor_name LOOP
  -- Statements using record_variable
END LOOP;

Example:

DECLARE
  CURSOR c_students IS
    SELECT name, course FROM students;
BEGIN
  FOR student_rec IN c_students LOOP
    DBMS_OUTPUT.PUT_LINE('Name: ' || student_rec.name || ', Course: ' || student_rec.course);
  END LOOP;
END;

Output:

Name: Gagan, Course: BCA
Name: Love, Course: BCA
Name: Kush, Course: BCA
Name: Ayush, Course: BBA
