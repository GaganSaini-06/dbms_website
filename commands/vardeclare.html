VARIABLE DECLARATION

Introduction: Variables in PL/SQL are used to store data temporarily during the execution of a program. They must be declared before use and have a specific data type. Constants can also be declared to store fixed values that do not change.

PL/SQL Data Types:

PL/SQL supports many data types such as:

  • NUMBER(p,s) – Numeric values with precision and scale
  • VARCHAR2(n) – Variable-length character strings
  • CHAR(n) – Fixed-length character strings
  • DATE – Date and time
  • BOOLEAN – TRUE, FALSE, or NULL

Declaring Variables:

variable_name datatype [:= initial_value];

Declaring Constants:

Use the CONSTANT keyword to declare constants.

pi CONSTANT NUMBER := 3.14159;

Using %TYPE:

The %TYPE attribute is used to declare a variable with the same data type as a column or another variable.

emp_name employees.name%TYPE;

Using %ROWTYPE:

The %ROWTYPE attribute is used to declare a record that represents an entire row of a table.

emp_record employees%ROWTYPE;

Scope and Visibility:

Variables declared inside a block are local to that block and cannot be accessed outside of it. Outer block variables can be accessed inside inner blocks unless shadowed by another declaration.

Assignments and Expressions:

You can assign values using := operator.

total := price * quantity;

Operator Precedence:

PL/SQL follows a specific order of precedence for evaluating expressions:

  1. Exponents (**)
  2. Multiplication and Division (*, /)
  3. Addition and Subtraction (+, -)
  4. Logical NOT
  5. Logical AND
  6. Logical OR

Example:

DECLARE
  name employees.name%TYPE;
  salary NUMBER := 30000;
  bonus CONSTANT NUMBER := 5000;
  net_salary NUMBER;
BEGIN
  net_salary := salary + bonus;
  DBMS_OUTPUT.PUT_LINE('Net Salary: ' || net_salary);
END;
