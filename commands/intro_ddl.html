INTRODUCTION TO DDL

Definition: DDL stands for Data Definition Language. It is used to define, create, and modify the structure of database objects such as tables, indexes, views, schemas, etc.

Purpose: DDL commands help in setting up and organizing the database structure where the actual data will be stored and manipulated.

Common DDL Commands:

  • CREATE – Creates new tables or database objects.
  • ALTER – Modifies the structure of an existing object.
  • DROP – Deletes objects from the database.
  • TRUNCATE – Removes all records from a table but not its structure.
  • RENAME – Renames a database object.

Example:

CREATE TABLE students (RollNo NUMBER, Name VARCHAR2(50));
