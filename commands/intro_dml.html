INTRODUCTION TO DML

Definition: DML stands for Data Manipulation Language. It is used to manage data within the objects created using DDL.

Purpose: DML commands allow users to add, retrieve, modify, and delete data stored in a database.

Common DML Commands:

  • INSERT – Adds new data into a table.
  • UPDATE – Modifies existing data in a table.
  • DELETE – Removes data from a table.
  • SELECT – Retrieves data from one or more tables (sometimes categorized under DQL).

Example:

INSERT INTO students (RollNo, Name) VALUES (101, 'Gagan');
