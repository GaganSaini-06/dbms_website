LEFT OUTER JOIN

Definition: LEFT OUTER JOIN returns all rows from the left table and the matched rows from the right table.
If no match is found, it still shows the left table's data and fills the right side with NULL.

Syntax:

SELECT columns FROM table1 LEFT OUTER JOIN table2 ON table1.common_column = table2.common_column;

Example:

SELECT s.name, m.marks FROM Students s LEFT OUTER JOIN Marks m ON s.student_id = m.student_id;
