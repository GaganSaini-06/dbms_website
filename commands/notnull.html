NOT NULL CONSTRAINT

Definition:The NOT NULL constraint ensures that a column cannot have a NULL (empty) value. It is used when you want to make sure a field always has data.

Syntax:

CREATE TABLE table_name ( column_name datatype NOT NULL );

Example:

CREATE TABLE student ( roll_no INT NOT NULL, name VARCHAR(50) NOT NULL );

Output:

mysql> CREATE TABLE temp_table (Rollno INT NOT NULL, Name VARCHAR(35) NOT NULL, Age INT);
Query OK, 0 rows affected (0.11 sec)

mysql> INSERT INTO temp_table VALUES (90, 'GAGAN', NULL);
Query OK, 1 row affected (0.07 sec)

mysql> INSERT INTO temp_table VALUES (NULL, 'YUVI', 23);
ERROR 1048 (23000): Column 'Rollno' cannot be null
