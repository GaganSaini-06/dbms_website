COMMAND

Definition:The BETWEEN operator in SQL is used to filter the result set within a range of values. It selects values between a minimum and maximum value (both are included).

Syntax:

SELECT * FROM table_name WHERE column_name BETWEEN value1 AND value2;

Example:

SELECT * FROM students WHERE Age BETWEEN 20 AND 22;

Output:
