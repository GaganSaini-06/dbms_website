IF-ELSE STATEMENT

Definition: The IF-ELSE statement in PL/SQL is used when you want to execute one block of code if a condition is true, and another block of code if the condition is false.

Syntax:

IF condition THEN
        statements;
ELSE
        statements;
END IF;

Example:

DECLARE
        num NUMBER := 25;
    BEGIN
        IF num > 0 THEN
           DBMS_OUTPUT.PUT_LINE('Positive number');
        ELSE
           DBMS_OUTPUT.PUT_LINE('Negative number or zero');
        END IF;
    END;

Output:

Positive number
