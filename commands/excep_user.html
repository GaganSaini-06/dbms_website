USER-DEFINED EXCEPTIONS

Definition: User-defined exceptions are custom exceptions created by the programmer for handling specific application errors that are not covered by predefined exceptions.

You declare them using the EXCEPTION keyword and raise them using the RAISE statement.

Steps to Use User-Defined Exceptions:

  • Declare the exception using the EXCEPTION keyword.
  • Raise the exception using the RAISE statement.
  • Handle it in the EXCEPTION block.

Syntax:

DECLARE
  exception_name EXCEPTION;
BEGIN
  -- some logic
  RAISE exception_name;
EXCEPTION
  WHEN exception_name THEN
    -- handling code
END;

Example:

DECLARE
  insufficient_balance EXCEPTION;
  balance NUMBER := 500;
  withdrawal NUMBER := 1000;
BEGIN
  IF withdrawal > balance THEN
    RAISE insufficient_balance;
  ELSE
    balance := balance - withdrawal;
  END IF;

  DBMS_OUTPUT.PUT_LINE('Withdrawal successful.');
EXCEPTION
  WHEN insufficient_balance THEN
    DBMS_OUTPUT.PUT_LINE('Error: Insufficient Balance!');
END;

Output:

Error: Insufficient Balance!
