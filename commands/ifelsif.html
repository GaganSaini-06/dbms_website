IF-ELSIF-ELSE STATEMENT

Definition: The IF-ELSIF-ELSE statement in PL/SQL is used when you have multiple conditions to evaluate. It checks each condition one by one and executes the block of the first condition that is true. If none of the conditions are true, it executes the ELSE block.

Syntax:

IF condition1 THEN
        statements;
ELSIF condition2 THEN
        statements;
ELSIF condition3 THEN
        statements;
ELSE
        statements;
END IF;

Example:

DECLARE
        marks NUMBER := 85;
    BEGIN
        IF marks >= 90 THEN
           DBMS_OUTPUT.PUT_LINE('Grade A');
        ELSIF marks >= 75 THEN
           DBMS_OUTPUT.PUT_LINE('Grade B');
        ELSIF marks >= 60 THEN
           DBMS_OUTPUT.PUT_LINE('Grade C');
        ELSE
           DBMS_OUTPUT.PUT_LINE('Grade D');
        END IF;
    END;

Output:

Grade B
