SIMPLE LOOP

Definition: The SIMPLE LOOP in PL/SQL repeatedly executes a block of statements until an EXIT condition is met. It must include an explicit EXIT or EXIT WHEN statement inside the loop to avoid infinite execution.

Syntax:

LOOP
        statements;
        EXIT WHEN condition;
END LOOP;

Example:

DECLARE
        i NUMBER := 1;
    BEGIN
        LOOP
            DBMS_OUTPUT.PUT_LINE('Value of i: ' || i);
            i := i + 1;
            EXIT WHEN i > 5;
        END LOOP;
    END;

Output:

Value of i: 1  
Value of i: 2  
Value of i: 3  
Value of i: 4  
Value of i: 5
