LIMIT CLAUSE

Definition:The LIMIT clause is used to restrict the number of rows returned by the query. It is very useful when you only want to see a specific number of records.

Syntax:

SELECT column1, column2 FROM table_name LIMIT number;

Example:

SELECT * FROM students LIMIT 5;

Output:
