EXPLICIT CURSOR

Definition: An Explicit Cursor is a cursor that is explicitly declared and controlled by the programmer for SELECT queries that return more than one row. It allows row-by-row processing in PL/SQL.

Steps to Use:

  1. Declare the cursor
  2. Open the cursor
  3. Fetch data row by row
  4. Close the cursor

Syntax:

DECLARE
  CURSOR cursor_name IS SELECT_statement;
  variable_name table_name.column_name%TYPE;
BEGIN
  OPEN cursor_name;
  LOOP
    FETCH cursor_name INTO variable_name;
    EXIT WHEN cursor_name%NOTFOUND;
    -- process each row
  END LOOP;
  CLOSE cursor_name;
END;

Example:

DECLARE
  CURSOR c1 IS SELECT name FROM students;
  v_name students.name%TYPE;
BEGIN
  OPEN c1;
  LOOP
    FETCH c1 INTO v_name;
    EXIT WHEN c1%NOTFOUND;
    DBMS_OUTPUT.PUT_LINE('Student Name: ' || v_name);
  END LOOP;
  CLOSE c1;
END;

Output:

Student Name: Gagan
Student Name: Love
Student Name: Kush
Student Name: Ayush
