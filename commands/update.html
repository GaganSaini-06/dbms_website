UPDATE COMMAND

Definition:The UPDATE command in SQL is used to modify or change the existing records in a table.

Syntax:

UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;

Example:

UPDATE students SET age = 20 WHERE name = 'Gagan';
