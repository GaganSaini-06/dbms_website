ROLLBACK COMMAND

Definition: The ROLLBACK command is used to undo all the changes made by the current transaction before the COMMIT is issued. It brings the database back to the last consistent state.
Use Case:
Used when an error occurs and you want to cancel all recent operations done in the transaction.

Syntax:

ROLLBACK;

Example:

START TRANSACTION;

INSERT INTO students (Rollno, Name, Age, Class, Semester) 
VALUES (30, 'Hardik Pandya', 29, 'BBA', 4);

ROLLBACK;

Output:

mysql> SELECT * FROM students;
+--------+-------+------+--------+
| Rollno | Name  | Age  | DeptID |
+--------+-------+------+--------+
|     11 | LOVE  |   21 |      2 |
|     18 | NONY  |   22 |      1 |
|     90 | GAGAN |   19 |      1 |
+--------+-------+------+--------+
3 rows in set (0.00 sec)

mysql> START TRANSACTION;
Query OK, 0 rows affected (0.00 sec)

mysql> INSERT INTO students VALUES (22, 'RAHUL', 20, 2);
Query OK, 1 row affected (0.00 sec)

mysql> ROLLBACK;
Query OK, 0 rows affected (0.07 sec)

mysql> SELECT * FROM students;

| Rollno | Name  | Age  | DeptID |
+--------+-------+------+--------+
|     11 | LOVE  |   21 |      2 |
