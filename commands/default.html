DEFAULT CONSTRAINT

Definition: The DEFAULT constraint is used to assign a default value to a column when no value is provided during insertion.
It helps in reducing the need to always input a value manually for that column.

Syntax:

CREATE TABLE table_name ( column1 datatype, column2 datatype DEFAULT default_value );

Example:

CREATE TABLE student ( roll_no INT PRIMARY KEY, name VARCHAR(50), city VARCHAR(50) DEFAULT 'Delhi' );
