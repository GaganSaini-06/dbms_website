CREATING VIEW

Definition:A VIEW in SQL is a virtual table based on the result of an SQL query. It does not store data physically, but you can use it like a table to simplify complex queries.

Syntax:

CREATE VIEW view_name AS SELECT column1, column2, ... FROM table_name WHERE condition;

Example:

CREATE VIEW young_players AS SELECT Name, Age FROM cricket_players WHERE Age < 35;
