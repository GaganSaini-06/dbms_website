SELF JOIN

Definition:A Self Join is a regular join where a table is joined with itself. It is useful when comparing rows within the same table.

Syntax:

SELECT A.column_name, B.column_name
FROM table_name A, table_name B
WHERE condition;

Example:

SELECT A.Name AS Student1, B.Name AS Student2, A.Course
FROM students A, students B
WHERE A.Course = B.Course AND A.Name != B.Name;

Output:
