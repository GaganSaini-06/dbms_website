CHECK CONSTRAINT

Definition:The CHECK constraint is used to limit the value that can be stored in a column. It ensures that the value satisfies a specific condition.

Syntax:

CREATE TABLE table_name ( column_name datatype CHECK (condition) );
CREATE TABLE table_name ( column1 datatype, column2 datatype, CHECK (column1 > 0 AND column2 < 100) );

Example:

CREATE TABLE student ( name VARCHAR(50), age INT CHECK (age >= 18) );

Output:

mysql> CREATE TABLE temp_table3 (Rollno INT , Name VARCHAR(35), Age INT CHECK (AGE >= 18));
Query OK, 0 rows affected (0.10 sec)

mysql> INSERT INTO temp_table3 VALUES (90, 'GAGAN', 19);
Query OK, 1 row affected (0.06 sec)

mysql> INSERT INTO temp_table3 VALUES (80, 'SAGAR', 17);
ERROR 3819 (HY000): Check constraint 'temp_table3_chk_1' is violated.
