DESCRIBE COMMAND

Definition: The DESCRIBE command in SQL is used to display the structure of a table, including column names, data types, and whether null values are allowed. It's helpful to understand the schema of an existing table.

Syntax:

DESCRIBE table_name;

Example:

DESCRIBE student;

Output:
