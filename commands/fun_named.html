NAMED FUNCTION

Definition: A named function in PL/SQL is a function that is explicitly created with a name and stored in the database. It can be called whenever needed using its name.

Named functions are reusable blocks of code that perform a specific task and return a value. These functions are stored permanently and can be accessed later.

Syntax:

CREATE OR REPLACE FUNCTION function_name (
  parameter1 [mode] datatype,
  parameter2 [mode] datatype
) RETURN return_datatype
IS
  -- Variable declarations
BEGIN
  -- Function logic
  RETURN value;
END;

Example:

CREATE OR REPLACE FUNCTION get_square(num IN NUMBER)
RETURN NUMBER
IS
  sq NUMBER;
BEGIN
  sq := num * num;
  RETURN sq;
END;

Calling the Named Function:

DECLARE
  n NUMBER := 6;
  result NUMBER;
BEGIN
  result := get_square(n);
  DBMS_OUTPUT.PUT_LINE('Square is: ' || result);
END;

Output:

Square is: 36
