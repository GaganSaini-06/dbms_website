ALTER COMMAND

Definition: The ALTER command in SQL is used to modify the structure of an existing table. It allows you to add, delete, or change columns.

Syntax:

FOR ADDING COLUMN:-
ALTER TABLE table_name ADD (column_name datatype, column_name datatype, ...);
FOR MODIFYING COLUMN:-
ALTER TABLE table_name MODIFY old_column_name new_datatype, MODIFY old_column_name new_datatype, ...;
FOR DROPPING COLUMN:-
ALTER TABLE table_name DROP column_name, DROP column_name, ...;

Example:

ALTER TABLE student ADD (Class varchar(35), Semester int);
ALTER TABLE student MODIFY Rollno varchar(10);
ALTER TABLE student DROP Semester;
