INTRODUCTION TO DQL

Definition: DQL stands for Data Query Language. It is used to fetch data from a database based on specific conditions.

Purpose: DQL is mainly focused on retrieving data from one or more tables. It does not perform any changes or modifications to the data.

Main DQL Command:

  • SELECT – Retrieves data from a database table or view.

Example:

SELECT * FROM students;
SELECT name, course FROM students WHERE age > 18;
