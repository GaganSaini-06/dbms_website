UNION

Definition: The UNION operator in SQL is used to combine the result of two SELECT queries and returns only unique rows (duplicates are removed by default).

Syntax:

SELECT column1, column2 FROM table1  
UNION  
SELECT column1, column2 FROM table2;

Note:
Both SELECT statements must have same number of columns,
And the columns should have compatible data types.

Example:

Query:    SELECT name FROM team1
          UNION
          SELECT name FROM team2;

team1 = {Rohit, Virat, Gill}
team2 = {Gill, Smith, Stokes}

Result (UNION) = {Rohit, Virat, Gill, Smith, Stokes}
Combines both tables, removes duplicates.
