PL/SQL BLOCK STRUCTURE

What is a PL/SQL Block? A PL/SQL program is structured into blocks. Each block forms a unit of code that can include declarations, execution logic, and error handling. These blocks can be nested inside each other.

There are three main sections of a PL/SQL Block:

  • Declaration Section: Used to declare variables, constants, cursors, etc. (optional)
  • Execution Section: Contains the main logic or SQL statements. (mandatory)
  • Exception Section: Handles runtime errors or exceptions. (optional)

Syntax:

DECLARE
  -- Variable declarations (optional)
BEGIN
  -- Main execution code (required)
EXCEPTION
  -- Error handling code (optional)
END;

Example:

DECLARE
  num NUMBER := 10;
BEGIN
  DBMS_OUTPUT.PUT_LINE('Number is: ' || num);
END;

Output:

Number is: 10
