INTRODUCTION TO OPERATORS

Definition: Operators in SQL are special symbols or keywords used to perform operations on data stored in tables. These are essential for creating conditions, filtering data, and performing calculations.

Types of SQL Operators:

  • Arithmetic Operators: Used to perform mathematical operations.
    • + (Addition)
    • - (Subtraction)
    • * (Multiplication)
    • / (Division)
    • % (Modulus)
  • Comparison Operators: Used to compare values.
    • =, != or <>, >, <, >=, <=
  • Logical Operators: Used to combine multiple conditions.
    • AND, OR, NOT
  • Special Operators: Used for special conditions.
    • BETWEEN, IN, LIKE, IS NULL

Example:

SELECT * FROM employees
WHERE salary > 30000 AND department IN ('HR', 'IT');
