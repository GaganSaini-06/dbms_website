INTRODUCTION TO TCL

Definition: TCL stands for Transaction Control Language. It is used to manage transactions in a database and control changes made by DML statements.

Purpose: TCL commands help in making sure the database remains consistent, even if errors occur during a series of operations.

Common TCL Commands:

  • COMMIT – Saves all the changes made in the current transaction.
  • ROLLBACK – Reverts the changes made in the current transaction.
  • SAVEPOINT – Sets a point within a transaction to which a rollback can be done.

Example:

INSERT INTO students VALUES (101, 'Aman', 20, 'BCA');
SAVEPOINT sp1;
DELETE FROM students WHERE Rollno = 101;
ROLLBACK TO sp1;
