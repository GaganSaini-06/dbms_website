LIKE OPERATOR

Definition: The LIKE operator in SQL is used to search for a specific pattern in a column using characters.

Syntax:

SELECT * FROM table_name WHERE column_name LIKE 'pattern';

Example:

SELECT * FROM students WHERE Name LIKE 'V%';
SELECT * FROM students WHERE Name LIKE '%an';
SELECT * FROM students WHERE Name LIKE '_u%';
