ORDER BY CLAUSE

Definition:The ORDER BY clause is used to sort the result of a query in ascending (default) or descending order based on one or more columns.

Syntax:

SELECT column1, column2 FROM table_name ORDER BY column1 [ASC | DESC];

Example:

SELECT name, marks FROM students ORDER BY marks DESC;
