INSERT COMMAND

Definition:The INSERT command in SQL is used to add new rows (records) into a table in a database.

Syntax:

INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);

Example:

INSERT INTO students (name, age) VALUES ('Gagan', 19);
