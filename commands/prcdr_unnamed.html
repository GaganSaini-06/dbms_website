UNNAMED PROCEDURE (ANONYMOUS BLOCK)

Definition: An Unnamed Procedure, also called an Anonymous Block, is a PL/SQL program block that does not have a name and is not stored in the database. It is written and executed directly without being saved permanently.

Key Features:

  • Not stored in the database
  • Useful for quick testing and one-time operations
  • Cannot be called like a procedure or function

Structure:

DECLARE
  -- Variable declarations
BEGIN
  -- Executable statements
EXCEPTION
  -- Exception handling statements
END;

Example:

DECLARE
  v_name VARCHAR2(20) := 'Gagan';
BEGIN
  DBMS_OUTPUT.PUT_LINE('Welcome, ' || v_name);
END;

Output:

Welcome, Gagan
