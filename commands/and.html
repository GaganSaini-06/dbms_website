AND OPERATOR

Definition: The AND operator in SQL is used to filter records based on more than one condition. It returns only those rows where **all conditions** are true.

Syntax:

SELECT * FROM table_name WHERE condition1 AND condition2;

Example:

SELECT * FROM students WHERE Age = 19 AND Course = 'BCA';
