IS OPERATOR

Definition:The IS operator in SQL is used to compare a value with NULL. You can't use = to check NULLs, so IS is used with NULL, NOT NULL, TRUE, or FALSE.

Syntax:

SELECT * FROM table_name WHERE column_name IS NULL;
OR
SELECT * FROM table_name WHERE column_name IS NOT NULL;

Example:

SELECT * FROM students WHERE Name IS NULL;
