INTERSECT

Definition: The INTERSECT operator is used to return only the common rows between two SELECT statements. It eliminates duplicates automatically and returns only the rows that appear in both result sets.

Syntax:

SELECT column1, column2 FROM table1  
INTERSECT  
SELECT column1, column2 FROM table2;

Note: Both SELECT statements must have the same number of columns and same data types in the same order.

Example:

Query:-  SELECT name FROM team1
         INTERSECT
         SELECT name FROM team2;

team1 = {Rohit, Virat, Gill}
team2 = {Gill, Smith, Stokes}

Result (INTERSECT) = {Gill}
Shows only common data in both tables.

Output:
