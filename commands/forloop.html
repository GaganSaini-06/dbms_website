FOR LOOP

Definition: The FOR LOOP in PL/SQL is used to execute a block of code a specific number of times. It automatically handles the initialization, condition-checking, and incrementing.

Syntax (IN):

FOR counter IN [REVERSE] start..end LOOP
        statements;
END LOOP;

Example (Forward Loop):

DECLARE
        i NUMBER;
    BEGIN
        FOR i IN 1..5 LOOP
            DBMS_OUTPUT.PUT_LINE('Value of i: ' || i);
        END LOOP;
    END;

Example (Reverse Loop):

DECLARE
        i NUMBER;
    BEGIN
        FOR i IN REVERSE 5..1 LOOP
            DBMS_OUTPUT.PUT_LINE('Reverse i: ' || i);
        END LOOP;
    END;

Output:

Value of i: 1  
Value of i: 2  
Value of i: 3  
Value of i: 4  
Value of i: 5  

Reverse i: 5  
Reverse i: 4  
Reverse i: 3  
Reverse i: 2  
Reverse i: 1
