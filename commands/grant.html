GRANT COMMAND

Definition: GRANT command is used to give specific privileges/permissions to users on database objects like tables, views, procedures, etc.

Syntax:

GRANT privilege_list ON object_name TO 'username'@'host';

privilege_list → Like SELECT, INSERT, UPDATE, etc.
object_name → Table or database on which permission is given
username → User who gets the access
host → Usually 'localhost'

Example:

GRANT SELECT, INSERT ON students TO 'gagan'@'localhost';

Meaning: The user gagan can now read (SELECT) and add data (INSERT) into the students table from localhost.

Output:

mysql> CREATE USER 'new_user'@'localhost' IDENTIFIED BY 'new_user123';
Query OK, 0 rows affected (0.10 sec)

mysql> GRANT SELECT, INSERT ON students TO 'new_user'@'localhost';
Query OK, 0 rows affected (0.09 sec)
