DROP COMMAND

Definition: The DROP command in SQL is used to permanently remove a database object like a table, database, view, or index from the database. Once dropped, all data and structure are lost and cannot be recovered.

Syntax:

DROP DATABASE database_name;
DROP TABLE table_name;

Example:

DROP DATABASE school;
DROP TABLE student;
