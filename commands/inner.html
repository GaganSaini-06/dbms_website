INNER JOIN

Definition: The INNER JOIN keyword selects records that have matching values in both tables.
It returns only the rows where there is a match in both tables.

Syntax:

SELECT table1.column1, table2.column2, ... FROM table1 INNER JOIN table2 ON table1.common_column = table2.common_column;

Example:

SELECT s.name, m.marks FROM Students s INNER JOIN Marks m ON s.student_id = m.student_id;
