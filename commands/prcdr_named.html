NAMED PROCEDURE

Definition: A Named Procedure in PL/SQL is a subprogram that performs a specific task, is stored in the database with a name, and can be reused multiple times by calling it.

Key Features:

  • Stored in the database permanently
  • Can be called from other PL/SQL blocks or applications
  • Supports parameters (IN, OUT, IN OUT)

Syntax:

CREATE OR REPLACE PROCEDURE procedure_name
IS
  -- Declaration section
BEGIN
  -- Executable section
END procedure_name;

Example:

CREATE OR REPLACE PROCEDURE greet_user
IS
  v_name VARCHAR2(20) := 'Gagan';
BEGIN
  DBMS_OUTPUT.PUT_LINE('Hello, ' || v_name);
END greet_user;

Calling the Procedure:

BEGIN
  greet_user;
END;

Output:

Hello, Gagan
