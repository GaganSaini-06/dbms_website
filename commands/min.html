MIN() FUNCTION

Definition:The MIN() function is used to find the smallest (minimum) value in a specified column. It works on numeric, date, or text data (alphabetically for text).

Syntax:

SELECT MIN(column_name) FROM table_name WHERE condition;

Example:

SELECT MIN(marks) FROM student_marks;
