SHOW COMMAND

Definition: The SHOW command in SQL is used to display information about databases, tables, columns, or other database objects. It helps in checking the structure and existing data elements in your database.

Syntax:

SHOW object_type;

Example:

SHOW DATABASES;
SHOW TABLES;

Output:
