CROSS OR CARTESIAN JOIN

Definition: A Cartesian Join returns the cross product of two tables — that means each row from the first table is combined with every row from the second table.
It doesn't require any ON condition.

Syntax:

SELECT * FROM table1, table2;
OR
SELECT * FROM table1 CROSS JOIN table2;

Example:

SELECT * FROM Students CROSS JOIN Courses;
