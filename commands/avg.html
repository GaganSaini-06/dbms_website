AVG() FUNCTION

Definition:The AVG() function is used to calculate the average value of a numeric column. It adds all the values and divides by the total number of values.

Syntax:

SELECT AVG(column_name) FROM table_name WHERE condition;

Example:

SELECT AVG(marks) FROM student_marks;
