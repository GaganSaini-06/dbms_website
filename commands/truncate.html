TRUNCATE COMMAND

Definition: The TRUNCATE command in SQL is used to remove all records from a table without deleting the table structure itself. It is faster than DELETE and cannot be rolled back in most databases because it does not log individual row deletions.

Syntax:

TRUNCATE TABLE table_name;

Example:

TRUNCATE TABLE student;

Output:

mysql> SELECT * FROM new_table;
