INTRODUCTION TO PROCEDURES

Definition: A Procedure in PL/SQL is a subprogram that performs a specific task. It is a named block that can take input (parameters), execute a set of statements, and optionally return values using OUT parameters.

Advantages:

  • Helps in modular programming
  • Increases code reusability
  • Simplifies debugging and maintenance
  • Improves performance through pre-compilation

Syntax:

CREATE OR REPLACE PROCEDURE procedure_name (
  param1 [IN | OUT | IN OUT] datatype,
  param2 [IN | OUT | IN OUT] datatype
) IS
  -- Declarations
BEGIN
  -- Procedure body statements
END procedure_name;

Example:

CREATE OR REPLACE PROCEDURE greet_student(name IN VARCHAR2) IS
BEGIN
  DBMS_OUTPUT.PUT_LINE('Hello, ' || name || '!');
END;

Calling the Procedure:

BEGIN
  greet_student('Gagan');
END;

Output:

Hello, Gagan!
