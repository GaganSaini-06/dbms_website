DROPPING VIEW

Definition:Dropping a view means permanently deleting a view from the database using the DROP VIEW statement. Once dropped, it cannot be used again unless recreated.

Syntax:

DROP VIEW view_name;

Example:

DROP VIEW young_players;

Output:

mysql> SELECT * FROM young_players2;
+---------------+------+
| Name          | Age  |
+---------------+------+
| Virat Kohli   |   35 |
| Rohit Sharma  |   36 |
| Hardik Pandya |   30 |
+---------------+------+
3 rows in set (0.00 sec)

mysql> DROP VIEW young_players2;
Query OK, 0 rows affected (0.07 sec)

mysql> SELECT * FROM young_players2;
ERROR 1146 (42S02): Table 'bca2ndyear_4090.young_players2' doesn't exist
