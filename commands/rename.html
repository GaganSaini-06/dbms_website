RENAME COMMAND

Definition: The RENAME command in SQL is used to change the name of an existing database object like a table. It helps you rename a table without changing its structure or data.

Syntax:

RENAME TABLE old_table_name TO new_table_name;

Example:

RENAME TABLE student TO student_data;

Output:

mysql> SHOW TABLES;
+---------------+
| Tables_in_new |
+---------------+
| new_table     |
+---------------+
1 row in set (0.00 sec)

mysql> RENAME TABLE new_table TO old_table;
Query OK, 0 rows affected (0.09 sec)

mysql> SHOW TABLES;
+---------------+
| Tables_in_new |
+---------------+
| old_table     |
+---------------+
1 row in set (0.00 sec)
