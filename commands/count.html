COUNT() FUNCTION

Definition:The COUNT() function is used to count the number of rows (records) in a table that match a specified condition. It helps in finding out how many entries are there.

Syntax:

SELECT COUNT(column_name) FROM table_name WHERE condition;

Example:

SELECT COUNT(roll_no) FROM student;
