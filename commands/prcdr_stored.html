STORED PROCEDURE

Definition: A Stored Procedure in PL/SQL is a precompiled block of code stored permanently in the database that can be executed when needed. It helps in reusability and better performance.

Advantages of Stored Procedures:

  • Improves performance due to precompilation
  • Reduces code duplication
  • Enhances security and integrity
  • Easy to maintain and modify

Syntax:

CREATE OR REPLACE PROCEDURE procedure_name
IS
  -- Declarations
BEGIN
  -- Executable statements
END procedure_name;

Example:

CREATE OR REPLACE PROCEDURE show_message
IS
BEGIN
  DBMS_OUTPUT.PUT_LINE('This is a stored procedure.');
END show_message;

Calling the Stored Procedure:

BEGIN
  show_message;
END;

Output:

This is a stored procedure.
