INTRODUCTION TO CLAUSES

Definition: Clauses in SQL are used to define the conditions and rules for retrieving or manipulating data from a database. They refine queries and make them more specific.

Commonly Used SQL Clauses:

  • WHERE: Filters records based on a condition.
  • ORDER BY: Sorts the result-set in ascending or descending order.
  • GROUP BY: Groups rows that have the same values into summary rows.
  • HAVING: Filters records after grouping (used with GROUP BY).
  • LIMIT / ROWNUM: Limits the number of records returned (database-specific).

Example:

SELECT course, COUNT(*)
FROM students
WHERE age > 18
GROUP BY course
HAVING COUNT(*) > 1
ORDER BY course;
