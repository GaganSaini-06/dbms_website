GOTO or JUMPING STATEMENT

Definition: The GOTO statement in PL/SQL is used to transfer the control unconditionally to a labeled statement or section in the program. It is generally avoided in modern programming as it can make code harder to read and maintain.

Syntax:

GOTO label_name;

<>
  statements;

Example:

DECLARE
  x NUMBER := 1;
BEGIN
  IF x = 1 THEN
    GOTO skip;
  END IF;

  DBMS_OUTPUT.PUT_LINE('This will not print');

  <<skip>>
  DBMS_OUTPUT.PUT_LINE('Skipped to here using GOTO');
END;

Output:

Skipped to here using GOTO
