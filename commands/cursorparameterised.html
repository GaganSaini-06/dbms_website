PARAMETERIZED CURSOR

Definition: A Parameterized Cursor allows you to pass parameters to a cursor at the time of opening it. It helps in making the cursor more dynamic by accepting values during execution.

Use: Helpful when you want to fetch data based on different conditions or filter values dynamically.

Syntax:

DECLARE
  CURSOR cursor_name(parameter datatype) IS
    SELECT columns FROM table_name WHERE column = parameter;

  variable table_name.column%TYPE;
BEGIN
  OPEN cursor_name(value);
  LOOP
    FETCH cursor_name INTO variable;
    EXIT WHEN cursor_name%NOTFOUND;
    -- process each row
  END LOOP;
  CLOSE cursor_name;
END;

Example:

DECLARE
  CURSOR c_course(cname VARCHAR2) IS
    SELECT name FROM students WHERE course = cname;
  v_name students.name%TYPE;
BEGIN
  OPEN c_course('BCA');
  LOOP
    FETCH c_course INTO v_name;
    EXIT WHEN c_course%NOTFOUND;
    DBMS_OUTPUT.PUT_LINE('Student Name: ' || v_name);
  END LOOP;
  CLOSE c_course;
END;

Output:

Student Name: Gagan
Student Name: Love
Student Name: Kush
